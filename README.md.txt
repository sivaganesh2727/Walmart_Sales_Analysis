# Walmart Sales Analysis - Power BI Dashboard

## 📌 Project Overview
This project is a **Sales Analysis Dashboard** created using **Power BI**, based on Walmart sales data. The dashboard provides valuable insights into sales performance, revenue trends, and key business metrics to help improve decision-making.

## 🚀 Features of the Dashboard
- **Data Import & Transformation:** Cleaned and structured raw Walmart sales data.
- **Interactive Visualizations:** Created sales trends, revenue breakdowns, and profit analysis.
- **KPIs & Insights:** Identified key performance indicators (KPIs) like total revenue, best-selling categories, and customer trends.
- **Filters & Drill-throughs:** Enabled filters for region, date, and product categories.
- **DAX Measures:** Used advanced DAX functions to calculate meaningful business metrics.

## 🛠️ Installation & Setup
### **1️⃣ Install Power BI**
Download and install **Power BI Desktop** from the official Microsoft website:
👉 [Download Power BI](https://powerbi.microsoft.com/)

Power BI installation Step by step Guide:https://learn.microsoft.com/en-us/power-bi/fundamentals/desktop-getting-started

### **2️⃣ Get the Dataset**
- Download the **Walmart Sales Dataset** (CSV/Excel format) from the project folder or online sources.
- Place the dataset inside the `/Data` folder.

### **3️⃣ Load Data into Power BI**
- Open **Power BI Desktop**.
- Click **Home > Get Data > Excel/CSV**.
- Select the dataset and load it into Power BI.

### **4️⃣ Data Transformation in Power Query**
- Cleaned null values, duplicate records, and formatted data types.
- Used **Power Query Editor** to structure data.
- Created **calculated columns** and **measures**.

### **5️⃣ Create Visualizations**
- Used **bar charts, line charts, pie charts, KPI cards, and slicers**.
- Applied filters and interactions for better insights.

### **6️⃣ Publish to Power BI Service**
- Click **Publish** in Power BI Desktop.
- Select **Workspace** and upload the report.
- Share via **Power BI Service**.
